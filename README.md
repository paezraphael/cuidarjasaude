🧡 CuidarJá

Um aplicativo que ajuda idosos a encontrar atendimento de saúde e transporte acessível de forma rápida, simples e segura.

📌 Problema

Muitas pessoas idosas enfrentam dificuldades no dia a dia para:

Encontrar o hospital ou unidade de saúde mais próxima 🏥
Saber qual o melhor caminho ou transporte para chegar até lá 🧭
Acessar opções de transporte adaptado (cadeirantes, mobilidade reduzida) ♿
Lidar com interfaces digitais complexas e pouco acessíveis 📱

Isso gera atrasos em atendimentos, desinformação e dependência de terceiros.

💡 Solução

O CuidarJá foi criado para simplificar esse processo.

O app permite:

📍 Encontrar hospitais e unidades de saúde próximas em tempo real
🗺️ Ver rotas simples e diretas até o local
♿ Indicar opções de transporte acessível
🚗 Futuramente: agendamento de transporte adaptado
🧑‍🦳 Interface simples, com botões grandes e leitura fácil
🎯 Objetivo

Criar uma solução digital simples, acessível e confiável para facilitar o acesso à saúde e mobilidade para idosos.

🚀 MVP (primeira versão)

Nesta fase inicial, o CuidarJá terá:

Mapa com localização do usuário 📍
Exibição de hospitais próximos 🏥
Rota até o destino 🧭
Interface otimizada para acessibilidade 👓
🛠️ Tecnologias
Flutter 📱
Google Maps API 🗺️
Geolocator 📍
Firebase (futuro backend) 🔥
🧭 Visão do projeto

O CuidarJá não é apenas um app — é uma ferramenta de inclusão digital e cuidado humano.

A ideia é evoluir para:

Integração com serviços públicos de transporte
Agendamento de vans acessíveis
Assistente por IA para ajudar idosos a usar o app
Alertas de saúde e emergência
