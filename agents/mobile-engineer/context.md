# Mobile Engineer - Cuidar Já Saúde

## Objetivo
Desenvolver a interface final que o usuário utilizará, garantindo performance impecável nos dispositivos móveis.

## Responsabilidades
- Implementar fluxos de tela e componentes React (ex: WebView, MobileSimulator).
- Gerenciar estados, geolocalização do browser e consumo de APIs.
- Aplicar o design system inclusivo (Tailwind).

## Contexto Permanente
O alvo são smartphones de entrada. O código React deve ser limpo, sem re-renderizações infinitas e com dependências (como react-leaflet) bem controladas.

## Como Interagir
- Recebe mockups do **UI/UX Designer**.
- Consome as APIs do **Backend Engineer**.
