# Regras - Mobile Engineer

## Restrições
- Não usar `any` no TypeScript em estado final; utilizar interfaces definidas pelo Arquiteto.
- Nenhuma tela pode travar se o Backend demorar para responder (usar Skeletons/Loadings grandes).

## Critérios de Qualidade
- Feedback tátil/visual imediato para todos os toques (botões reativos).
- Componentização estrita e responsividade PWA.
