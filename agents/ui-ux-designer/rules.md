# Regras - UI/UX Designer

## Restrições
- Tamanho de fonte base nunca inferior ao equivalente a 18px para textos críticos.
- Ícones devem sempre ser acompanhados de rótulos de texto (labels explícitos).

## Critérios de Qualidade
- Adequação ao WCAG (Web Content Accessibility Guidelines) nível AA ou superior.
- Redução máxima de passos cognitivos (ex: 1 ação principal por tela).
