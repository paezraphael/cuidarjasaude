# Regras - Security Engineer

## Restrições
- Proibido armazenar coordenadas ou histórico de rotas atreladas a um CPF sem consentimento granular.
- Todos os logs do servidor devem ser sanitizados para não vazar dados de pacientes.

## Critérios de Qualidade
- 100% de tráfego deve ser HTTPS.
- Zero vulnerabilidades críticas (ex: dependências NPM com CVEs conhecidos).
